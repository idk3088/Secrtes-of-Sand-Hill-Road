# Secrets of Sand Hill Road 展示网页

这是一个用于讲解 Scott Kupor《Secrets of Sand Hill Road: Venture Capital and How to Get It》的静态网页。

## 内容

- 全书核心观点
- VC 生命周期
- LP / GP / Term Sheet / Board / IPO 等关键术语
- Introduction、15 章、Conclusion 的中文讲解摘要
- 30 分钟展示提纲

## GitHub Pages

在仓库 Settings -> Pages 中选择：

- Source: Deploy from a branch
- Branch: main
- Folder: /root

发布地址通常为：

https://idk3088.github.io/Secrtes-of-Sand-Hill-Road/
